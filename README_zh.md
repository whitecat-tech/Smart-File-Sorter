# 智能文件整理器 v2.0 🚀

[English](./README.md) | [🇨🇳 简体中文]

一款基于 Python 开发的现代化桌面端文件管理工具。只需一键，即可瞬间自动整理凌乱的文件夹，并精准清除重复文件。现已全面升级原生图形界面（GUI）与密码级文件去重算法。

---

## ✨ 核心特性

- **原生桌面图形界面**: 使用 `tkinter` 构建，彻底告别黑框框命令行。内置“浏览文件夹”按钮，纯鼠标点击操作。
- **智能去重（核心卖点）**: 采用 **MD5 密码级哈希算法**（8KB 分块流式读取），通过二进制底层内容比对文件属性。**即使两个文件名字完全不同，只要内容一样，就能精准抓取。**
- **安全隔离**: 重复文件会被安全地移入独立的 `/duplicates` 归类夹中，绝不盲目误删用户数据。

---

## 🛠️ 支持格式

| 分类 | 支持的后缀格式 | 目标文件夹名 |
| :--- | :--- | :--- |
| 🎵 **音频** | `.mp3`, `.flac`, `.wav`, `.m4a` | `/audio` |
| 🎬 **视频** | `.mp4`, `.mkv`, `.avi`, `.mov` | `/video` |
| 🖼️ **图片** | `.jpg`, `.jpeg`, `.png`, `.gif` | `/picture` |
| 👯 **重复文件** | *文件底层内容完全相同* | `/duplicates` |
